# projekti-inxhinieri-softverike

# Emri i Projektit

City Motion – Website për Autosallon

# Përshkrimi i Projektit

City Motion është një webaplikacion i krijuar për prezantimin e një autosalloni dhe automjeteve që ofron për shitje. Platforma u mundëson vizitorëve të shohin ofertat e makinave, shërbimet e autosallonit, lajmet dhe të dërgojnë kërkesë për test drive ose interes për një automjet.

# Çfarë bën aplikacioni

* Shfaq automjetet dhe ofertat speciale të autosallonit
* Prezanton shërbimet që ofron kompania
* Paraqet lajme dhe artikuj rreth industrisë së automjeteve
* Lejon përdoruesit të dërgojnë kërkesë për kontakt ose rezervim
* Ofron informacion rreth autosallonit dhe kontaktit

# Teknologjitë e përdorura

* HTML – për strukturën e faqes
* CSS – për dizajnin dhe stilizimin
* JavaScript – për funksionalitetet interaktive
* Font Awesome – për ikonat

# Teknologjitë që planifikoj të përdor në të ardhmen

* MySQL – për ruajtjen e të dhënave të automjeteve dhe kërkesave të klientëve
* Bootstrap ose CSS Grid/Flexbox – për dizajn responsive
* Hosting dhe Domain – për publikimin e projektit online
